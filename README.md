# Horoscope Script

A simple JavaScript script that maps a birth month to a zodiac sign and prints a random fortune.

## Usage

1. Open `horoscope.js`.
2. Set the `month` variable to your birth month.
3. Run the script with Node:

```bash
node horoscope.js
```

## Notes

- This example uses only the birth month, not the full birth date.
- For a more accurate zodiac sign, you would use the day and month together.
